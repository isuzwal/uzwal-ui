## WhileHover Effect
 ```
 <motion.button 
  whileHover={{
        rotateX:20,
        rotateY:12,
        boxShadow:"0px 10px 60px rgba(249, 115, 22, 0.7)",
        y:-5
      }}
      whileTap={{
        y:10
      }}
      style={{
        translateZ:100
      }}
      transition={{
        duration:0.3,
        ease:"easeIn"
      }}
 >
 </motion.button>
 ```
