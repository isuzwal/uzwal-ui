## Overview

This project is a collection of UI design experiments and learning exercises. It covers topics such as animations, styling, and Figma-inspired layouts. The goal is to document progress and improve understanding of modern UI development.

## Features

- Custom animations
- Responsive layouts
- Theming and styling examples
- Figma-inspired components

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/practice-ui.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
